# eqaa
Title stands for "Equalize Amino Acids", sounds a little grand but it's actually just strin goperations
(which actually is bioinformatics ina a nutshell).

It started life as a challenge from the Nuritas (peptides) company during summer 2017, presented as a webpage and required you to give the length
of shortest possible subsequence of a 800 amino acid string within which letter replacements could be undertaken so that the entire string ended up
having equal abundances of all 20 amino acids.

I sent in an answer and the app promptly told me I was wrong (an automatic answer).

I was a bit mystified and did some more work with it. However it was one of my speed coding efforts
and though I was able to resurface the code in 2026, it was pretty hard to understand what I was trying to do.

This is actually a typical process, due to the immersive nature of coding these things.
